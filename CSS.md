# CSS Cheat sheet 

### CSS RULES
```
Selector{
  Property:Value;
  }
```

### TYPES OF CSS RULES

internal
```
<head>
  <style type="text/css">
    <!--style rule-->
  </style>
</head>
```

external
```
<head>
  <link type="stylesheet" href="path/filename.css"/>
</head>
```
inline 
```
<[html_element] style="property:value;"></[html_element]>
```

#### color names and codes
EXAMPLE for red color
```
<!--COLOR NAMES-->
color: red;
<!--COLOR CODE-->
color: #FF0000
<!--HEX CODE-->
color: rgb(255,0,0)

```

##### common properties
```
  //backgroundcolor:
  background: [color]  // example : green
  
  //letter fonts:
  font-family: [fontname]  // example : arial
  
  //text color:
  color: [color]       // example: red
  
```
